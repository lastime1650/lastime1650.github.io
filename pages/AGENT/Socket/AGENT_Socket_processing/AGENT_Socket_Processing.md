# **커널에서 중앙서버의 요청을 처리하는 방법을 설명합니다.**

### 에이전트와 중앙서버간 초기통신이 이루어지면, 에이전트는 항상 Receive상태가 됩니다.  그렇기 때문에 중앙서버의 요청이 있기 전까지 데이터를 전달하지 않는 형식입니다.
